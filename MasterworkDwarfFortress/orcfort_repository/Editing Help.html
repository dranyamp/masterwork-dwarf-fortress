Editing Help

If you think the Orc Fortress Manual is missing something, or just needs a little clean-up, then you can help! The manual is automatically generated from the Orc Fortress Manual Wiki, which you might be looking at right now. All you need is a free github account to make edits.

The Orc Fortress Manual is written using Markdown, the same language that's used for reddit posts, github commits, and in numerous other places on the Internet. It's a very intuitive syntax, so don't be afraid to just start editing a page and pick it up as you go.

While the github wiki supports formats other than markdown, please only use markdown for the manual. There are some back-end tools which rely upon it, and they won't convert other formats.

The wiki uses Gollum, a lightweight wiki engine that runs on top of git.

As a guide, the following links may prove useful, for both markdown syntax and gollum documentation itself:

  • http://daringfireball.net/projects/markdown/syntax
  • https://github.com/gollum/gollum/wiki

Please be bold with your edits, and don't be afraid to make changes.
